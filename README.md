# SPLIS KOREA Homepage

SPLIS KOREA - Engineering-First Integration

## Live Demo
[https://sidefill.github.io/splis-korea-homepage/](https://sidefill.github.io/splis-korea-homepage/)

## Project Structure
- `index.html`: Main landing page
- `product.html`: Products page with video and slideshow features
- `style.css`: Main stylesheet (includes responsive design)
- `script.js`: Interactive elements (Slideshow, Video, etc.)
- `assets/`: Images and video resources

## Features
- **Responsive Design**: Optimized for desktop and mobile.
- **Product Video**: Autoplay video integration with fallback poster.
- **Hero Slideshow**: Automatic background image cycling on product page.
- **Geominimalist Design**: Custom patterns and glassmorphism effects.
